Voidwalker - performance analysis for Apex Legends players

Project Voidwalker is an open-source Python application designed for Apex Legends players who want to track their in-game performance and extract valuable data for analysis. By continuously analyzing the screen during matches, the program can detect useful information and compile it into a single JSON file for further processing. Currently, the app tracks game modes, legends picked by each teammate, and end-of-game stats such as kills and damage. The program was initially developed for personal use only and has some limitations, but future plans include making it more robust and convenient to use.
